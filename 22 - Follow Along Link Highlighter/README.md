#### 取得物件的 size 與位置資料

這邊使用 [getBoundingClientRect()](https://developer.mozilla.org/en-US/docs/Web/API/Element/getBoundingClientRect) 這個方法，  
它會回傳物件的 size 與它的位置的資料，  
這樣我們想讓物件 highlight 時，  
高光處會跟著物件大小而有所不同，就可以使用這個方法。
